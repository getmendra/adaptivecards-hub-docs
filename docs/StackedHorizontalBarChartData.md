Defines the collection of data series to display in as a stacked horizontal bar chart.

* * *

Schema

| Name | Type | Default | Description | In version |
| --- | --- | --- | --- | --- |
| data | Array ofobject | The data points in the series. Valid values: \`StackedHorizontalBarChartDataPoint\` | 1.5 |
| title | string | The title of the series. | 1.5 |
| key | string | Defines an optional key for the object. Keys are seldom needed, but in some scenarios, specifying keys can help maintain visual state in the host application. | 1.0 |
